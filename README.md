This project focuses on using DAX (Data Analysis Expressions) in Microsoft Power BI Desktop to create calculated measures for data analysis and reporting. The goal is to transform raw data into meaningful insights through dynamic calculations.

⸻

Objectives

* Create calculated measures using DAX
* Perform aggregations and calculations
* Implement conditional logic
* Enhance report interactivity
* Support data-driven insights

⸻

Tools & Technologies

* Power BI Desktop
* DAX (Data Analysis Expressions)

⸻

DAX Functions Used

* SUM – To calculate total values
* COUNT – To count records
* IF – For conditional logic


⸻

Key Features

* Dynamic measures that update based on filters
* Improved accuracy in calculations
* Interactive dashboards powered by DAX logic

⸻

Insights

* DAX enables deeper analysis beyond basic visuals
* Measures respond dynamically to user interaction
* Proper use of DAX improves reporting efficiency
